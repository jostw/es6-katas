# ES6 Katas

ES6 kata practices from http://es6katas.org/

## Installation

```
npm install
```

## Test

```
npm test
```

## Development

Checkout to the branch **katas** for practicing.

```
git checkout katas
```

My fixes with the katas are in the commits with message ``` fix xxx.js ```. 
Just show it if you can't get it right!
